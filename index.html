OSPF Design: Core to Edge

Standardized OSPF Area 1 configuration for distribution and edge blocks.

Architecture Overview

Our design uses a Totally Stubby Area configuration for Area 1 to minimize routing table overhead on edge devices while maintaining full reachability.

  • Core-1: Acts as the ABR, injecting a default route (Type 3 LSA) into Area 1.
  • CP-BR-1: Transit point for Area 1.
  • Edge Routers: Passive by default; minimal configuration required.

Core-1 (ABR) Config

router ospf 1
 area 1 stub no-summary
 default-information originate always

Edge Router Config

router ospf 1
 area 1 stub
 !
interface GigabitEthernet1/0/1
 ip ospf 1 area 1
 ip ospf network point-to-point
